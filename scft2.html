2D SCFT Simulation: AB diblock + A + B homopolymers









Progress:


The image above shows the A monomer concentration for an AB diblock copolymer melt + A and B homopolymers. You can modify the input parameters: xN, f and L (system length, in units of Re), average A and B homopolymer concentrations (pA and pB - if they do not sum to 1, copolymer fills the rest) and the length radio of homopolymers to copolymers (Nh/N). You can draw on the image to change the state of the system. The converge button (partially) converges the system using SCFT. It only runs a few steps. To converge more, press the button again.

Note that the numerical stability depends on the number of steps along the chain (which is fixed) and the system resolution (which changes with L) so it may become unstable. Also note that the progress is given as a fraction of the max number of steps, not a measure of the actual convergence.

You can open the console to show more detail in the convergence progress, specifically the root mean squared 'error' in the A and B fields. The convergence algorithm that I implemented is a slow one, so it may be... slow.
